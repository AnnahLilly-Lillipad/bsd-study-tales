# 📚 Bungou Study Tales

An AI-powered study game inspired by gacha systems and character collection games, featuring Bungou Stray Dogs-themed cards, study battles, and AI-generated learning content.

---

# 🌟 Overview

Bungou Study Tales turns studying into a game.

Import your study material, let AI generate learning activities, earn rewards, collect character cards, build teams, and battle your way through exams.

The goal is to make studying engaging while rewarding consistent learning through progression systems, collection mechanics, and strategy.

---

# ✨ Core Features

## Study System

- Import PDFs
- Import websites/URLs

- Paste text directly
- AI-generated study materials

- Study streaks

### Quiz

### Flashcard Flip

Review terms and definitions.

### Memory Match

Match concepts and keywords.

### Fill-in-the-Blank Story

Complete missing information within generated passages.

### Falling Terms

### Boss Battle Exam

Study questions become combat attacks.

### Spaced Repetition

Long-term memory training.
